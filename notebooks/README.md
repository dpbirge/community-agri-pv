# Notebooks

Jupyter notebooks for interactive analysis, testing, and exploration of the Community Agri-PV simulation model.

## Current Notebooks

- `testing.ipynb` - Interactive testing and validation of model components

## Usage

Notebooks can be used to:
- Test data generation scripts
- Validate scenario configurations
- Explore pre-computed datasets
- Run simulations and visualize results
- Compare water policy outcomes
- Generate custom visualizations and reports
